# Extended Attributes

### View a files extended attributes

```
xattr <filename>
```

<figure><img src="../.gitbook/assets/xattr.png" alt=""><figcaption></figcaption></figure>

### Delete an extended attribute

**-r** recursive

**-d** delete

```
xattr -r -d <attribute> <filename>
```
